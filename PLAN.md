## Design

### Color Palette
| Role | Hex | Description |
|------|-----|-------------|
| Background | `#0c1a12` | Very dark forest green |
| Surface | `#152b1a` | Slightly lighter for cards |
| Border | `#1f3e27` | Subtle section dividers |
| Accent | `#4fa872` | Mid-green for highlights/links |
| Accent Hover | `#72c493` | Lighter on hover |
| Text Primary | `#edf4ef` | Off-white, slightly green-tinted |
| Text Muted | `#7aaa8a` | Secondary text, labels |

### Fonts (Google Fonts)
- **Headings**: `Playfair Display` — elegant serif, adds personality to the minimal layout
- **Body**: `Inter` — clean, readable, modern

### Visual Theme
- Single-page, centered layout, generous whitespace
- Minimal decorative elements — let the copy carry the personality
- Subtle hover animations on fact cards
- Mobile responsive
